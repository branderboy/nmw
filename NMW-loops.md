# NMW Viral Loop System

## Core Goal

Turn every artist signup into distribution, attendance growth, and new artist acquisition without relying on ads.

The website should create self-growing loops through artist behavior, team sharing, event reminders, and referrals.

---

# 1. Artist Registration Team Loop

## Flow

Artist signs up to perform at NMW.

After signup, prompt them to invite:

* manager
* DJ
* producer
* videographer
* friends
* street team

Each invite should:

* connect back to the event
* encourage RSVP
* push attendees into the ecosystem

## Goal

One artist should naturally bring multiple people into the platform.

## Feature Requirements

* Generate unique referral/share link for each artist
* Simple invite/share UI
* Track how many people joined through each artist
* Mobile-first sharing experience

---

# 2. Artist Promo Funnel Loop

## Flow

After artist registration:

* generate branded confirmation page
* generate artist promo card/flyer
* generate personalized event link

Prompt artist to:
“Share your appearance at New Music Wednesdays.”

## Goal

Artists market the event for us through their own audience.

## Feature Requirements

* Auto-generated share image
* Instagram story optimized graphic
* Copy/paste share caption
* Share buttons for Instagram/X/Facebook

---

# 3. Google Calendar Event Loop

## Flow

After RSVP or artist registration:
prompt user to:
“Add NMW to Google Calendar.”

## Goal

Keep NMW visible weekly and create recurring attendance behavior.

## Feature Requirements

* Google Calendar integration
* “Add to Calendar” button
* Recurring Wednesday event support
* Mobile optimized

---

# 4. Invite 5 Artists Loop

## Flow

Artist signs up to perform.

Then sees:
“Invite 5 artists and unlock free entry.”

## Goal

Artists recruit more artists into the ecosystem.

## Tier Structure

### Invite 3 Artists

Unlock:

* priority line access

### Invite 5 Artists

Unlock:

* free ticket

### Invite 10 Artists

Unlock:

* interview consideration
* featured placement
* priority performance consideration

## Feature Requirements

* Referral tracking system
* Invite counter
* Reward unlock system
* Artist referral dashboard
* Email/share link support

---

# 5. Main Viral Engine

## System Loop

Artist signs up →
shares with team/fans/artists →
new people RSVP/register →
more artists perform →
more content/moments created →
more artists want access →
repeat

---

# UX Direction

Keep everything:

* mobile first
* fast
* social
* clean
* culturally authentic

Avoid:

* fake gamification
* cluttered dashboards
* unnecessary widgets
* overly corporate UX

The experience should feel like:
exclusive artist access + culture + momentum.

---

# Priority Build Order

1. Artist signup flow
2. Referral/share system
3. Invite tracking
4. Google Calendar integration
5. Auto-generated promo assets
6. Reward unlock system

# Important

The system should encourage:

* sharing
* artist recruitment
* recurring attendance
* social posting
* audience growth

without feeling spammy or gimmicky.
