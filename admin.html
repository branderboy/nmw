Invite to NMW

Upload a list of DJs, artists, or labels and queue invitations. Each recipient gets an audience-specific landing link and is added to the right pipeline.

Parsed
0 recipients ready, DJ Members
How it works
  1. Pick the audience, each gets a different invite template + landing page.
  2. Drop a CSV/TXT or paste a list. We parse names + emails + phones.
  3. Click Queue Invites. Each recipient gets an audience-specific email + WhatsApp template (when opted-in).
  4. Recipients land on a custom join URL, opt-in to The Blast, and drop into the right pipeline (DJ / Artist / Label).
Pre-launch note Invites are stored locally for now. When the migration lands (Phase 6 Resend + Phase 8 WhatsApp), queued batches will fire automatically.
